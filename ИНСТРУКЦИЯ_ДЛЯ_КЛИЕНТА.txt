1. КАК ЗАПУСТИТЬ ПРОГРАММУ
================================================================================

1. Найдите на рабочем столе или в папке программы файл "Запустить CRM.bat"
2. Дважды кликните по нему
3. Появится чёрное окно - НЕ ЗАКРЫВАЙТЕ ЕГО!
4. Автоматически откроется браузер с программой

ВАЖНО: Пока работаете с программой, чёрное окно должно быть открыто!
       Чтобы выключить программу - закройте чёрное окно.

================================================================================
2. РЕГИСТРАЦИЯ И ВХОД
================================================================================

ПЕРВЫЙ ЗАПУСК (регистрация):
1. Нажмите "Регистрация"
2. Введите:
   - Логин (придумайте, например: admin)
   - Пароль (минимум 8 символов, не только цифры)
   - Повторите пароль
   - Название вашего бизнеса
3. Нажмите "Зарегистрироваться"

ПОСЛЕДУЮЩИЕ ВХОДЫ:
1. Введите свой логин
2. Введите пароль
3. Нажмите "Войти"

3. РАБОТА С КЛИЕНТАМИ
================================================================================

ДОБАВИТЬ КЛИЕНТА:
1. Нажмите "Клиенты" в меню слева
2. Нажмите зелёную кнопку "Добавить"
3. Заполните:
   - Имя клиента
   - Телефон
   - Заметки (необязательно)
4. Нажмите "Сохранить"

НАЙТИ КЛИЕНТА:
1. Перейдите в "Клиенты"
2. В строке поиска введите имя или телефон
3. Нажмите кнопку поиска (лупа)

РЕДАКТИРОВАТЬ КЛИЕНТА:
1. Найдите клиента в списке
2. Нажмите на его имя
3. Нажмите "Редактировать"
4. Измените данные
5. Нажмите "Сохранить"

УДАЛИТЬ КЛИЕНТА:
1. Откройте карточку клиента
2. Нажмите красную кнопку "Удалить"
3. Подтвердите удаление

ВНИМАНИЕ: При удалении клиента удаляются все его заявки!

================================================================================
4. РАБОТА С ЗАЯВКАМИ
================================================================================

ДОБАВИТЬ ЗАЯВКУ:
1. Откройте карточку клиента
2. Нажмите "Новая заявка"
3. Заполните:
   - Услуга (что делаете)
   - Статус (Новая / В работе / Завершена)
   - Цена (необязательно)
4. Нажмите "Сохранить"

ИЗМЕНИТЬ СТАТУС ЗАЯВКИ:
1. Откройте заявку
2. Нажмите "Редактировать"
3. Выберите новый статус
4. Нажмите "Сохранить"

ДОБАВИТЬ КОММЕНТАРИЙ К ЗАЯВКЕ:
1. Откройте заявку
2. Внизу страницы введите текст комментария
3. Нажмите "Добавить комментарий"

ПОСМОТРЕТЬ ВСЕ ЗАЯВКИ:
1. Нажмите "Заявки" в меню слева
2. Используйте фильтры: Все / Новые / В работе / Завершённые

================================================================================
5. ЭКСПОРТ В EXCEL
================================================================================

ЭКСПОРТ КЛИЕНТОВ:
1. Перейдите в "Клиенты"
2. Нажмите зелёную кнопку "Excel"
3. Файл скачается автоматически

ЭКСПОРТ ЗАЯВОК:
1. Перейдите в "Заявки"
2. Нажмите зелёную кнопку "Excel"
3. Файл скачается автоматически

================================================================================
6. СМЕНА ЯЗЫКА
================================================================================

1. В левом меню под названием бизнеса есть кнопки RU и TM
2. Нажмите RU для русского языка
3. Нажмите TM для туркменского языка

================================================================================
7. РЕЗЕРВНОЕ КОПИРОВАНИЕ (ОЧЕНЬ ВАЖНО!)
================================================================================

Все ваши данные хранятся в файле: db.sqlite3

ДЕЛАЙТЕ КОПИЮ РЕГУЛЯРНО:
1. Закройте программу (закройте чёрное окно)
2. Найдите файл db.sqlite3 в папке программы
3. Скопируйте его на флешку или в другую папку
4. Рекомендуется делать копию каждую неделю

ВОССТАНОВЛЕНИЕ ИЗ КОПИИ:
1. Закройте программу
2. Замените файл db.sqlite3 на сохранённую копию
3. Запустите программу

================================================================================
8. РЕШЕНИЕ ПРОБЛЕМ
================================================================================

ПРОБЛЕМА: Программа не запускается
РЕШЕНИЕ: 
- Убедитесь что Python установлен
- Попробуйте перезагрузить компьютер

ПРОБЛЕМА: Браузер не открывается автоматически
РЕШЕНИЕ:
- Откройте браузер вручную
- Введите адрес: http://127.0.0.1:8000

ПРОБЛЕМА: Забыли пароль
РЕШЕНИЕ:
- Обратитесь к специалисту который устанавливал программу

ПРОБЛЕМА: Данные пропали
РЕШЕНИЕ:
- Восстановите из резервной копии (см. раздел 7)
